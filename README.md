# 杨顺超 - 个人简历与作品展示博客

这是一个为前端/跨端开发工程师**杨顺超**量身定制的个人简历与作品展示博客。项目采用轻量、极速的现代原生网页技术栈（HTML5 + CSS3 + ES6 JavaScript）构建，支持完美的响应式体验、明暗主题切换、项目分类筛选及动态过渡效果，且能够直接零配置部署在 Vercel 托管平台上。

---

## 🌟 项目特色

- **极简原生技术栈**：不依赖 Webpack/Vite 等繁重的编译打包工具，直接双击 `index.html` 即可在浏览器中完美展现，秒开级加载速度。
- **现代化科技风美学**：默认采用高格调暗黑风（Cyber-Minimalism）视觉设计，配合霓虹渐变边框与高质感玻璃拟态（Glassmorphism）卡片；内置一键明暗主题无缝切换。
- **丰富的微交互动效**：
  - 首屏搭载动态打字机循环展示核心技能。
  - 基于 `IntersectionObserver` API 实现的流式滚动显现动效。
  - 作品卡片支持基于分类的无感切换过滤。
  - 完备的交互式项目详情模态弹窗（Modal），展示各项目的职责与技术指标。
- **便携联系功能**：内置一键复制手机与邮箱至剪贴板功能，并配备精美流畅的 Toast 提示。
- **SEO & 语义化友好**：包含完整的 Meta 标签设置与规范的 HTML5 语义化节点结构。

## 💻 本地运行

1. **直接打开**：
   在文件管理器中双击 `index.html`，即可在任何现代浏览器中直接预览运行。

2. **使用本地轻量服务器运行（推荐）**：
   如果您安装了 Node.js，可以在项目根目录下执行以下命令以获得更好的热更新体验：
   ```bash
   # 使用 npx 快速运行静态服务器
   npx serve .
   ```
   或者使用 VS Code 中的 **Live Server** 插件直接运行。

## ⚡ Vercel 部署步骤

该项目已配置就绪，可实现零成本一键部署：

### 方法一：通过 GitHub 关联导入（推荐）
1. 将当前项目上传至您的个人 **GitHub** 仓库中。
2. 登录 [Vercel 官网](https://vercel.com/)。
3. 点击 **Add New -> Project**，导入您刚才上传的 GitHub 仓库。
4. 框架预设（Framework Preset）选择 **Other**，Build Command 与 Output Directory 均保留为空（Vercel 会自动识别 `index.html` 作为静态页面）。
5. 点击 **Deploy**，大约 10 秒后即可获得免费的公网访问域名（支持绑定您在 Namesilo 购买的个人域名）。

### 方法二：使用 Vercel CLI 本地命令行部署
1. 在命令行中全局安装 Vercel CLI：
   ```bash
   npm install -g vercel
   ```
2. 进入项目根目录下，执行部署命令并登录绑定：
   ```bash
   vercel
   ```
3. 按照提示选择配置（全部回车默认即可），部署完成后将自动复制预览链接，再次执行 `vercel --prod` 即可发布到正式生产域名。
